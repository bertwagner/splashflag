# Splashflag

SplashFlag is an IoT device for notifying my neighbors when we are going swimming.

If you want to build this device for yourself, follow the READMEs in each subfolder.

A full breakdown of features, functionality, and additional build steps can be found at https://bertwagner.com/posts/splashflag-building-an-iot-swimming-notification-device-from-scratch/#how-to-build-your-own-splashflag .
